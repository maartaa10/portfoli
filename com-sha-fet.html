Com s'ha fet

Aquesta pàgina s'ha creat amb una estructura HTML senzilla i funcional, amb l'objectiu de presentar el portafoli en forma d'un Escape Room interactiu i amb una estètica inspirada en Matrix.

Estructura HTML

He organitzat el codi HTML en seccions clares i diferenciades per facilitar la navegació i la comprensió:

  • Encapçalament: Inclou el nom i el rellotge, creat amb JavaScript, per afegir un toc de dinamisme a la pàgina.
  • Seccions: Cada pàgina està dividida en seccions que s'auto-oculten o es desbloquegen en resoldre cada enigma. Això es fa amb una combinació de classes CSS i JavaScript.
  • Footer: Un footer simple amb enllaços a les xarxes socials i a la pàgina d'inici, en coherència amb el disseny Matrix.

Estil Matrix amb CSS

Per aconseguir l’estètica Matrix, he aplicat diversos elements visuals amb CSS:

  • Colors: Fons negre amb text verd (#00FF00) per simular la pantalla de codi de Matrix.
  • Tipografia: He utilitzat una font monoespai, 'Courier New', per donar l’aparença de codi.
  • Animacions: Hi ha un efecte de "màquina d'escriure" en el text de les instruccions i un efecte hover per als botons. També s'usa una animació per fer aparèixer el rellotge.
  • Classes específiques: He creat classes CSS com .section-content per estilitzar cada secció de manera consistent i amb un toc de misteri.

Funcionalitat amb JavaScript

He utilitzat JavaScript per afegir interactivitat i fer que l'experiència sigui més immersiva:

  • Rellotge: Un script de JavaScript actualitza l'hora en temps real a l'encapçalament. Utilitza la funció setInterval() per refrescar cada segon i mantenir l’usuari immers en l'ambient.
  • Enigmes: He creat funcions que generen enigmes amb condicions que l'usuari ha de resoldre per desbloquejar cada secció.
  • Efecte de "màquina d'escriure": Amb un bucle que recorre cada lletra de les instruccions, el text apareix gradualment, donant una sensació de misteri.
  • Control de seccions: Quan l'usuari resol correctament un enigma, es canvien les classes CSS de les seccions per mostrar-les o amagar-les segons calgui.
  • Musica: S'ha afegit la BSO de Mision Imposible, que sona quan es comença a jugar per a oferir a l'usuari una experiencia més agradable

Enigmes i Interactivitat

Els enigmes estan dissenyats per ser senzills però entretinguts, creant una experiència interactiva per a l’usuari:

  • Enigma de "Sobre Mi": Cal respondre quants mesos tenen 28 dies.
  • Enigma de "Projectes": Es demana convertir '0011' de binari a decimal.
  • Enigma de "Skills": Cal calcular el nombre de segons en una hora.
  • Enigma de "Experiencia": Cal endevinar quin numero es descriu.
  • Enigma de "Contacte": Cal resoldre quin numero es l'unic igual a la suma de les seves xifres.
